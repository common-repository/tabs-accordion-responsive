Tabs Accordion Responsive is the easiest and lightest way (No JS/Only CSS) to add a set of tabs to your site. It’s as simple as choosing a layout, selecting your settings and filling your tabs data.

== Description ==

= Build your set of tabs. =

This plugin gives you the possibility to build your tabs, select a pre-built layout and display it thanks to a shortcode.
You’ll be able to dynamically add or delete a tabs to your set, move it and insert your data.
The front-end output is only using the CSS language, there is no Javascript that could slow down your website.

In order to be responsive, your Tabs will switch to an Accordion style on small screen.

It is currently not possible to have several sets of tabs on the same page.

= How to use our plugin ? =

* You just have to download your plugin and upload it to your "wp-content/plugins" folder.
* Activate your plugin on your Dashboard and you will see a Tabs Accordion tabs.
* At this point, you just have to create a new Tabs Accordion by completing the fields.
* Once created, go back to the list of your Tabs Accordion and you will see a Shortcode.
* All you have to do, is put the Shortcode wherever you want to get your Tabs.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/tabs-accordion-responsive` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress

== Usage ==
1. Simply use the new tab in the admin panel named Tabs Accordion.
2. Create your first item.
3. Go back to the item's list and grab the shortcode.
4. Insert the shortcode wherever you want.
